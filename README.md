# NatOptim
## Nature-Inspired Optimization Library for C++

# Install

Install CMake
'''
$ sudo apt-get install cmake
'''
Clone this git repository in your $HOME directory
'''
git clone https://github.com/mkbhs/natoptim.git
'''
Build NatOptim:
'''
$ cd natoptim
$ mkdir build
$ cd build
$ cmake ..
$ make
''
